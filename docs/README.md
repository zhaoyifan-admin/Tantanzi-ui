### 快速上手

```bash
# 安装
yarn add  @zhaoyifannan/tantanzi-ui -S # 或者：npm i @zhaoyifannan/tantanzi-ui -S

# 引入
import Avue from '@zhaoyifannan/tantanzi-ui';
import '@zhaoyifannan/tantanzi-ui/lib/tantanzi-ui.css';
Vue.use(Avue);

```
