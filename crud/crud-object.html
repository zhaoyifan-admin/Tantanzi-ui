# Object对象形式

TIP

2.10.12+

TIP

之前的版本option中column是数组的格式，2.10.12+后支持对象格式，操作更加方便

option中的column可以配置成对象形式，用prop作为key

<avue-crud :option="option" :data="data" ></avue-crud>

<script>
export default {
  data () {
    return {
      form: {},
      data: [{
        name: '姓名',
        sex: 14
      }],
      option: {
        labelWidth: 120,
        column: {
          name: {
            label: '姓名',
          },
          sex: {
            label: '年龄',
          }
        }
      }
    }
  },
  created () {
    this.option.column.name.label += '(name)';
  }
}
</script>
